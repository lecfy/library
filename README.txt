Requirements

PHP 7.2 or higher, MySQL PDO

Installation

1. Unpack zip archive in private directory (not accessible from the web)
2. Move the contents of "public" folder to your public/web directory, for e.g. public_html and change path to your app/Config.php in public/index.php
3. Create a database and user with access permissions and import library.sql
4. Configure HOST (website url) and database details in app/Config.php
5. Go to your website
